# 项目文档

## 项目结构
本项目采用标准的Python项目结构，主要包含以下目录：

- `src/`: 源代码目录
  - `core/`: 核心功能模块
    - `file_utils.py`: 文件处理工具
  - `main.py`: 主程序入口
- `docs/`: 文档目录
  - `development.md`: 开发指南
  - `README.md`: 文档主页
- `examples/`: 示例代码目录
  - `demo.py`: 演示程序
- `tests/`: 测试用例目录
  - `test_main.py`: 主程序测试

## 快速开始
请参考 `examples/demo.py` 了解如何使用本项目。

详细的开发指南请参考 `docs/development.md`。
